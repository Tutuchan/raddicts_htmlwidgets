morrisjs

Line charts

library(morrisjs)
morrisjs(mdeaths) %>% 
  mjsLine()

Bar charts

morrisjs(mdeaths) %>% 
  mjsBar()

Multiple series

morrisjs(ts.union(fdeaths, mdeaths)) %>% 
  mjsLine()

Options

Options are available on the morris.js github page and are passed as a list through the options parameter.

morrisjs(ts.union(fdeaths, mdeaths)) %>% 
  mjsLine(options = list(pointSize = 1, postUnits = " deaths"))
